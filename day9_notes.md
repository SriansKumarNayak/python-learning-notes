## 📌 String Operations

- Strings can be manipulated using operators

### Example:
```python
text = "Hello"

print(text * 2)   # HelloHello
print(text + " World")  # Hello World
```

---

📌 Loop Practice
### Example:
```python
# Print numbers from 1 to 10
for i in range(1, 11):
    print(i)

# Print reverse numbers
for i in range(10, 0, -1):
    print(i)
```

📌 Basic Problems
1. Sum of Numbers
```python
total = 0

for i in range(1, 6):
    total += i

print(total)   # 15
```

---

2. Factorial
```python
num = 5
fact = 1

for i in range(1, num + 1):
    fact *= i
print(fact)   # 120
```

---


3. Count Characters
```python
text = "python"

count = 0
for char in text:
    count += 1

print(count)
```
