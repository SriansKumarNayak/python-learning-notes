## 📌 Lists

- Ordered collection of elements
- Mutable (can be changed)

### Example:
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)

## 📌 Tuples
- Immutable (cannot be changed)
my_tuple = (1, 2, 3)
print(my_tuple[0])

## 📌 Dictionaries
- Key-value pairs
student = {
    "name": "Aman",
    "marks": 90
}
print(student["name"])


numbers = [10, 20, 30]
numbers.append(40)

info = {"city": "Delhi"}
info["country"] = "India"

print(numbers)
print(info)
